Приложение для организации рабочего дня.

Аннотация

Назначение:
 Решение проблемы планирования дня.
Целевая аудитория:
 Работающие, учащиеся люди.

Варианты использования
Сценарии использования

С – специалист.
ВП – wеб-приложение.

Работа со словарем предметной области

  1. С переходит в раздел для работы со словарем терминов.
  2. ВП отображает раздел для работы со словарем терминов.
  3. С вводит/корректирует название предметной области.
  4. С вводит/корректирует название словаря терминов.
  5. С вводит/корректирует содержимое словаря терминов.
  6. С нажимает «Сохранить».
  7. ВП сохраняет словарь терминов.

Прохождение теста

  1. С переходит в раздел для прохождения тестов.
  2. ВП отображает раздел для прохождения тестов.
  3. С нажимает «Запустить».
  4. ВП подготавливает список вопросв и ответов.
  5. ВП отображает вопрос и несколько вариантов ответа.
  6. С выбирает один из вариантов ответа.
  7. С нажимает «Далее».
  8. ВП отображает следующий вопрос и варианты ответа.
  9. С нажимает «Завершить».
  10. Вариант использования «Просмотр результатов прохождения теста».

Просмотр результатов прохождения теста

  1. ВП отображает сколько было вопросов в тесте.
  2. ВП отображает сколько было дано правильных ответов.
  3. ВП отображает процент правильных ответов.
  4. ВП отображает список вопросов с ответами, выбранными в процессе тестирования.
  5. ВП маркирует вопросы, на которые был дан правильный ответ.

Просмотр списка терминов

  1. С нажимает «Отобразить список терминов».
  2. ВП отображает список терминов.
  3. ВП отображает для каждого термина его описание.
  4. [С нажимает «Скрыть список терминов».]
  5. [ВП скрывает список терминов.]

Установка у термина отметки «Изучил»

  1. С знакомится с термином и его описанием, изучает дополнительные материалы.
  2. С нажимает «Изучил» для данного термина.
  3. ВП скрывает термин из списка терминов.

Сброс отметки «Изучил» у всех терминов

  1. С нажимает «Снять отметку «Изучил» у всех терминов».
  2. ВП запрашивает подтверждение на снятие отметки «Изучил» у всех терминов.
  3. С нажимает «Ок».
  4. ВП снимает отметку «Изучил» у всех терминов.
  5. ВП отображает список всех терминов.

Просмотр статистики

  1. С нажимает «Отобразить список терминов».
  2. ВП отображает общее количество терминов.
  3. ВП отображает количество изученных терминов.
  4. ВП отображает количество терминов для изучения.
  5. ВП отображает процент терминов для изучения.
  6. С нажимает «Изучил» у одного из терминов.
  7. ВП увеличивает на единицу количество изученных терминов, уменьшает на единицу количество терминов для изучения, пересчитывает процент терминов для изучения.

Сохранение списка терминов в файл в табличном формате

  1. С нажимает «Скачать список терминов».
  2. ВП предлагает выбрать папку для сохранения файла.
  3. С выбирает папку.
  4. С нажимает «Сохранить».
  5. ВП сохраняет список терминов в файл с расширение xls.

Поиск дополнительной информации

  1. С нажимает на кнопку поиска для конкретного термина в списке терминов.
  2. ВП осуществляет поиск данных в интернете.
  3. ВП отображает результаты поиска.
  4. С знакомится с результатами поиска.

Ввод описания терминов в режиме изучения терминов

  1. С нажимает «Отобразить список терминов в режиме редактирования».
  2. ВП отображает список терминов.
  3. ВП отображает для каждого термина его описание.
  4. С корректирует описание.
  5. С нажимает «Сохранить».
  6. ВП сохраняет описание термина.

Сохранение данных в файл

  1. С указывает имя файла.
  2. С нажимает «Скачать».
  3. ВП предлагает выбрать папку для сохранения файла.
  4. С выбирает папку.
  5. С нажимает «Сохранить».
  6. ВП сохраняет данные в файл.

Загрузка данных из файла

  1. С нажимает «Загрузить».
  2. ВП предлагает выбрать файл.
  3. С выбирает файл.
  4. С нажимает «Открыть».
  5. ВП запрашивает подтверждение на загрузку данных из файла.
  6. С нажимает «Ок».
  7. ВП загружает данные из файла.
