Приложение для организации рабочего дня.

Аннотация

Назначение:
 Приложение позволяет заменить ряд приложений с аналогичными возможностями, но не содержащие весь перечисленный функионал, что в конечном счете облегчает синхронизацию и перенос между устройствами.

Целевая аудитория:
 Учащиеся, работающие люди, которым необходимо планировать своё время.

Автор:
 Бондаренко Артём

Варианты использования


Сценарии использования

П – пользователь.
ВП – wеб-приложение.

Работа с заметками

  1. П переходит в раздел для работы со списком заметок.
  2. ВП отображает раздел для работы со списком заметок.
  3. П редактирует список заметок.
  4. ВП сохраняет список заметок по мере изменения.
  5. ВП перерисовывает раздел для работы со списком заметок.

Работа с задачами

  1. П переходит в раздел для работы с задачами.
  2. ВП отображает раздел для работы со списком заметок.
  3. П добавляет/удаляет задачи.
  4. ВП сохраняет список задач по мере изменения.
  5. ВП перерисовывает раздел для работы со списком заметок.
  6. П изменяет состояние задачи.
  7. ВП сохраняет состояние задач по мере изменения.
  8. ВП перерисовывает раздел для работы со списком заметок.

Работа с погодой

  1. П переходит в раздел для работы с прогнозом погоды.
  2. ВП отображает раздел для работы с прогнозом погоды.
  3. П нажимает на кнопку отображения прогноза по текущему местоположению.
  4. П предоставляет разрешение на использование локации.
  5. ВП отображает прогноз погоды на сегодня и на 7 дней по местоположению.

Загрузка данных из файла

  1. П нажимает «Upload Json».
  2. ВП предлагает выбрать файл.
  3. П выбирает файл.
  4. П нажимает «Открыть».
  5. ВП запрашивает подтверждение на загрузку данных из файла.
  6. П нажимает «Ок».
  7. ВП загружает данные из файла
  8. ВП отрисовывает раздел в соответствии с загруженными данными.

Сохранение данных в файл

  1. П указывает имя файла.
  2. П нажимает «Скачать».
  3. ВП предлагает выбрать папку для сохранения файла.
  4. П выбирает папку.
  5. П нажимает «Сохранить».
  6. ВП сохраняет данные в файл.

Просмотр прогноза по выбранному городу

  3. П вводит название города в строке поиска.
  4. ВП отображает прогноз погоды на сегодня и на 7 дней по выбранному городу.

Добавление задачи

  1. П переходит в раздел для работы с задачами.
  2. ВП отображает раздел для работы со списком задач.
  3. П заполняет поле новой задачи.
  4. П нажимает "Add"
  5. ВП сохраняет список задач по мере добавления.
  6. ВП перерисовывает раздел для работы со списком задач.

Изменение задачи

  1. П переходит в раздел для работы с задачами.
  2. ВП отображает раздел для работы со списком заметок.
  3. П заполняет поле новой задачи.
  4. П нажимает "Edit"
  5. ВП добавляет поле для изменения текста задачи.
  6. П изменяет значение в поле задачи
  7. П нажимает "Edit"
  8. ВП сохраняет список задач по мере добавления.
  9. ВП перерисовывает раздел для работы со списком заметок.

Изменение состояния задачи

  1. П переходит в раздел для работы с задачами.
  2. ВП отображает раздел для работы со списком заметок.
  3. П добавляет/снимает отметку с задачи.
  4. ВП сохраняет список задач по мере изменения состояний.
  5. ВП перерисовывает раздел для работы со списком заметок.

Просмотр активных задач

  1. П переходит в раздел для работы с задачами.
  2. ВП отображает раздел для работы со списком задач.

Просмотр выполненных задач

  1. П переходит в раздел для работы с задачами.
  2. ВП отображает раздел для работы со списком задач.

Редактирование заметки

  1. П переходит в раздел для работы со списком заметок.
  2. ВП отображает раздел для работы со списком заметок.
  3. П нажимает "View" у выбранной заметки.
  4. ВП отображает содержимое заметки в соседнем поле просмотра.
  5. П изменяет значение содержимого заметки.
  6. П нажимает "Save"
  7. ВП сохраняет изменение заметки.
  8. ВП перерисовывает раздел для работы со списком заметок.

Просмотр выбранной заметки

  1. П переходит в раздел для работы со списком заметок.
  2. ВП отображает раздел для работы со списком заметок.
  3. П нажимает "View" у выбранной заметки.
  4. ВП отображает содержимое заметки в соседнем поле просмотра.

Просмотр списка заметок

  1. П переходит в раздел для работы с заметками.
  2. ВП отображает раздел для работы со списком заметок.

Удаление конкретной заметки

  1. П переходит в раздел для работы с заметками.
  2. ВП отображает раздел для работы со списком заметок.
  3. П нажимает "Delete" у выбранной заметки.
  4. ВП сохраняет изменение списка заметок.
  5. ВП перерисовывает раздел для работы со списком заметок.

Удаление всех заметок

  1. П переходит в раздел для работы с заметками.
  2. ВП отображает раздел для работы со списком заметок.
  3. П нажимает "Delete All".
  4. ВП удаляет все элементы списка заметок.
  5. ВП перерисовывает раздел для работы со списком заметок.
Сценарии взаимодействия

Us – User.
Br – Browser.
DS – Data Storage.
API - API openweathermap.

Вход в систему

  1. Us просит Br запустить web-приложение.
  2. Br отображает навигационную панель с разделами для работы с заметками, задачами, погодой и разделом для просмотра информации о приложении.
  3. Br отображает раздел для работы со списком заметок.
  4. Br отображает раздел для работы с выбранной заметкой.
  5. Br запрашивает у DS список заметок.
  6. Br загружает список заметок в раздел для работы со списком заметок.
  7. Br запрашивает у DS список занятий.
  8. Br обновляет данные в разделе для работы со списком заметок.

Работа с заметками

  1. Us работает со списком заметок.
  2. Br просит DS сохранить список заметок.
  3. Br запрашивает у DS список заметок.
  4. Br отправляет DS обновленный список заметок.
  5. Br обновляет данные в разделе для работы со списком заметок.

Работа с задачами

  1. Us работает со списком задач.
  2. Br просит DS сохранить список заметок.
  3. Br запрашивает у DS список заметок.
  4. Br отправляет DS обновленный список заметок.
  5. Br обновляет данные в разделе для работы со списком заметок.

Работа с погодой

  1. Br отрисовывает текущую дату.
  2. Us предоставляет разрешение на работу с местоположением.
  3. Br получает данные о местоположении.
  4. Br запрашивает у API данные о прогнозе погоды.
  5. API отправляет данные о прогнозе.
  6. Br отрисовывает прогноз погоды на сегодня.
  7. Br отрисовывает прогноз погоды на 7 дней.

Загрузка данных из файла

  1. Us просит Br загрузить данные из файла.
  2. Br просит Us выбрать файл.
  3. Us выбирает файл с выгруженными ранее данными.
  4. Br получает данные из файла.
  5. Br просит DS удалить текущие данные.
  6. Br просит DS сохранить данные, полученные из файла.
  7. раздел в соответствии с загруженными данными

Сохранение данных в файл

  1. Us просит Br сохранить данные в файл.
  2. Br подготавливает информацию для сохранения.
  3. Br генерит имя файла.
  4. Br просит DS предоставить данные для выгрузки в файл.
  5. Br помещает данные в файл.
  6. Br просит Us откорректировать имя файла или принять предложенное.
  7. Br просит Us выбрать папку для сохранения файла.
  8. Us корректирует имя файла или принимает предложенное.
  9. Us выбирает папку.
  10. Br сохраняет файл в выбранную папку.

Просмотр прогноза по выбранному городу

  1. Br отрисовывает текущую дату.
  2. Us вводит название города в строке поиска.
  3. Br запрашивает у API данные о прогнозе погоды по городу.
  4. API отправляет данные о прогнозе.
  5. Br отрисовывает прогноз погоды на сегодня.
  6. Br отрисовывает прогноз погоды на 7 дней.

Добавление задачи

  1. Us просит Br создать новую задачу.
  2. Br просит DS добавить новую задачу.
  3. Br обновляет данные в разделе для работы со списком задач.

Изменение задачи

  1. Us просит Br изменить задачу.
  2. Br изменяет отображение задачи, чтобы можно было редактировать значение.
  3. Us просит Br сохранить изменение
  4. Br просит DS обновить значения.
  5. Br обновляет данные в разделе для работы со списком задач.

Изменение состояния задачи

  1. Us просит Br изменить состояние задачи.
  2. Br просит DS обновить состояние выбранной задачи.
  3. Br обновляет данные в разделе для работы со списком задач.

Редактирование заметки

  1. Us просит Br отобразить заметку.
  2. Br отображает значение содержимого задачи в соседнем поле просмотра.
  3. Us изменяет содержимое заметки.
  4. Us просит Br сохранить изменение
  5. Br просит DS обновить значения.
  6. Br обновляет данные в разделе для работы со списком заметок.

Просмотр выбранной заметки

  1. Us просит Br отобразить заметки.
  2. Br отображает значение содержимого задачи в соседнем поле просмотра.

Удаление конкретной заметки

  1. Us просит Br удалить заметку.
  2. Br просит DS удалить заметку.
  3. DS обновляет список заметок.
  4. Br перерисовывает раздел для работы со списком заметок.

Удаление всех заметок

  1. Us просит Br удалить все заметки.
  2. Br просит DS удалить все заметки.
  3. DS очищает список заметок.
  4. Br перерисовывает раздел для работы со списком заметок.
